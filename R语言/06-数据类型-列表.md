# R语言<数据类型-列表>

## 一、列表声明

```R
'''
list(...)
'''
# v1 <- 1:5; 
'''
m2 <- matrixclass(
	c(1:6, letters[1:6]), 
	nrow=3,  
	byrow=T, 
	dimnames=list(c("r1", "r2", "r3"), c("c1", "c2", "c3", "c4"))
)
'''
l1 <- list(com1=v1, com2=m2)
'''
print(l1)
$com1
[1] 1 2 3 4 5

$com2
   c1  c2  c3  c4 
r1 "1" "2" "3" "4"
r2 "5" "6" "a" "b"
r3 "c" "d" "e" "f"
'''
```

> 注意：列表的元素可以是不同的数据类型。

## 二、列表属性

```R
# 列表元素个数
length(l1)		# 2

# 列表名称
names(l1)		# ["com1", "com2"]
```

## 三、列表索引

```R
# 返回某元素, 返回类型：向量
li$com1		# 1, 2, 3, 4, 5

# 返回某元素，返回类型：矩阵
l1[["com2"]]
'''
   c1  c2  c3  c4 
r1 "1" "2" "3" "4"
r2 "5" "6" "a" "b"
r3 "c" "d" "e" "f"
'''

# 返回某元素，返回类型：矩阵
li[[2]]
'''
   c1  c2  c3  c4 
r1 "1" "2" "3" "4"
r2 "5" "6" "a" "b"
r3 "c" "d" "e" "f"
'''

# 返回某元素，返回类型：列表
li[1]		# 1, 2, 3, 4, 5

# 返回某元素，返回类型：列表
li["com1"]	# 1, 2, 3, 4, 5
```

> 注意点：

```R
# l1$com1、l1[["com2"]]、l1[[2]], 这三种会返回元素本身；
# l1[1], li["com1"], 这两种,返回的类型是list
```

## 四、列表操作

#### 4.1 列表新增元素

```R
l1$com3 <- 3:6
’‘’

```
