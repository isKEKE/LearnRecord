## 二、声明向量

#### 2.1 整数型向量

```R
v1 <- 1:5				# [1, 2, 3, 4, 5]

# 通过c函数生成
va <- c(v1, 3, 2, 7, 4, 6)		# [1, 2, 3, 4, 5, 3, 2, 7, 4, 6]
v2 <- c(3, 2, 7,4, 6)			# [3, 2, 7, 4, 6]

# 通过rep函数复制已有向量对象的N次为新的对象。
'''
rep(x, times, each)
x:			要复制的对象
times:		复制对象的复制次数
each:		每一个元素复制的次数
注意：each的优先级大于times的优先级，若它们同时都有参数，那么会先执行each相关逻辑，然后再执行times相关逻辑。
'''
v3 <- rep(v2, times=2)			# [3, 2, 7, 4, 6, 3, 2, 7, 4, 6]
v4 <- rep(v2, each=2)			# [3, 3, 2, 2, 7, 7, 4, 4, 6, 6]
v5 <- rep(v2, times=2, each=2)	# [3, 3, 2, 2, 7, 7, 4, 4, 6, 6, 3, 3, 2, 2, 7, 7, 4, 4, 6, 6]

# 通过seq生成向量
'''
seq(from, to, by, length.out, ...)
from:		生成向量起始值
to:			生成向量结束值
by:			步长，默认1
length.out:	期望生成的向量对象的长度, 如何是分数会被四舍五入。
'''
# 特殊点: seq(from=1, to=9)	  		  # [1, 2, 3, 4, 5, 6, 7, 8, 9]; 包含最后9
v6 <- seq(from=2, to=9, by=3)			# [2, 5, 8]; Python range?
v7 <- seq(from=2, to=9, length.out=3)	# [2, 5.5, 9]
v8 <- seq(from=2, by=3, length.out=3)	# [2, 5, 8]
```

#### 2.2 字符型向量

```R
# 通过c函数生成字符型向量
v9 <- c("aic", "bic", "cp")			# ["aic", "bic", "cp"]

# rep 复制
rep(v9, each=2)						# ["aic", "aic", "bic", "bic", "cp", "cp"]
```

#### 2.3 布尔值类型

```R
# c函数生成
c(T, T, F, T)		# [TRUE, TRUE, FALSE, TRUE]
```

#### 2.4 注意：不同类型组成向量

> ==会强制转换成同一类型==

```R
# 会强制转换成同一类型
c(T, "123")		# ["TRUE", "123"]
c(T, 1)			# [1, 1]
c(1, "abc")		# ["1", "abc"]
```

#### 2.5 向量名称

```R
# v2 [3, 2, 7, 4, 6]
names(v2)			# NULL
names(v2) <- v9		
'''
v9: ["aic", "bic", "cp"]
aic	bic	cp	NA	NA
3	2	7	4	6
'''
```

#### 2.6 向量长度

```R
length(v9)			# 3
# 可以通过赋值改变向量对象的长度
length(v9) <- 2		# ["aic", "bic"]
```

#### 2.7 向量索引

> 在R语言中，向量的起始索引是`1`。

```R
# v8 <- [2, 5, 8]
v8[3]				# 8
v8[c(1, 3, 5)]		# [2, 5, NA]

# 注意-c(1, 4) => [-1, -4]
# 这里意思表示取值排除索引为[1, 4]的值
# 所以结果[2, 5, 8] => [5, 8]
v8[-c(1, 4)]		# [5, 8]

# 通过向量名称取值
v2[c("aic", "cp")]	# [3, 7]

# 通过表达式取值; %% 取模运算
# 这里v1 %% 2 == 1 =>	[T, F, T, F, T]
v1[v1 %% 2 == 1]	# [1 3, 5]
```
