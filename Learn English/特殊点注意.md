#### 一、单词单数和复数的转换

| 序号 | 单词-单数 | 复数    |      | 序号 | 单词-单数 | 复数       |
| ---- | --------- | ------- | ---- | ---- | --------- | ---------- |
| 1    | dress     | dresses |      | 5    | man       | men        |
| 2    | box       | boxes   |      | 6    | woman     | women      |
| 3    | watch     | watches |      | 7    | housewife | housewives |
| 4    | dish      | dishes  |      | 8    | knife     | knives     |

- 注意点一：序号1-4，后面不是单词；
- 注意点二：序号=8，以`f`或`fe`结尾，转复数去掉`f`或`fe`添加`ves`。
